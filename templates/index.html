Data Sorter

Enter full file paths on disk, choose columns, arrange order, remove duplicates, and save the output directly to any folder. Supports CSV, TXT, and Excel for small files.

1. Choose input

Click Choose file to open the Windows file picker (CSV, TXT, or Excel), or paste a full path and click Load file. Excel is limited to 100,000 rows and 25 MB — use CSV or TXT for large files.

Click Choose files, then hold Ctrl and click each file to select many at once. Supported: CSV, TXT, Excel (.xlsx, .xls). You can also paste paths below (one per line).
